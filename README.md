# Spring Sensor Monitoring Clone Study

Spring Boot 기반 백엔드 아키텍처와 실시간 데이터 처리 구조를 학습하기 위해  
**실시간 센서 모니터링 시스템을 참고한 클론 코딩 스터디**를 진행하고 있습니다.

이 레포지토리는 **스터디 진행 과정에서 학습한 코드와 구조를 정리하는 공간**이며,

# Study Goal

단순히 Spring API를 만드는 수준이 아니라,

**Spring 기반 백엔드 위에서 다양한 외부 기술을 어떻게 연결하고 활용할 수 있는지 이해하는 것**을 목표로 합니다.

특히 다음과 같은 역량을 학습하는 것을 목표로 합니다.

- Spring Boot 기반 백엔드 아키텍처 이해
- 도메인 중심 구조 설계
- 실시간 데이터 처리 구조
- 이벤트 기반 시스템 설계
- 외부 메시징 시스템 연동
- 실시간 알림 시스템 구현
- 비동기 처리 및 확장 가능한 시스템 구조 이해

즉,

> **Spring을 중심으로 Kafka, WebSocket 등 다양한 외부 기술을 실제 서비스 구조에서 어떻게 활용하는지 학습하는 것이 목표입니다.**

# Study Plan (Overview)

스터디는 약 **8주 계획**으로 진행하며  
Spring 기반 시스템에 다양한 기술을 점진적으로 결합하는 방식으로 진행합니다.

### Week 1
Spring Boot 기반 프로젝트 구조 이해  
기본 도메인 설계 및 API 구조 학습

- Spring Boot
- JPA / ORM
- REST API 설계

---

### Week 2
이벤트 및 로그 기반 시스템 구조 이해

- 데이터 기록 구조
- 상태 관리
- 이벤트 로그 처리

---

### Week 3
실시간 데이터 처리 시스템 학습

- Kafka 기반 메시징

- 비동기 이벤트 흐름 이해

---

### Week 4
실시간 알림 시스템 구현

### Week 5 ~ 8
